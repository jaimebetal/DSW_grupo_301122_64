Presentación

Estimados estudiantes sean bienvenidos al curso Diseños de Sitios Web Cod: 301122. Este curso pertenece al programa de Ingeniería de Sistemas de la UNAD y hace parte de su componente electivo. El curso nos ofrece una iniciación básica en el campo del diseño web utilizando sistemas de control de versiones como GIT y la comunidad en línea GITHUB con su cliente para escritorio. Además, se utilizarán los lenguajes de diseño web HTML5 y CSS3 para el diseño de páginas web. En el siguiente video se hará una pequeña introducción al curso de Diseños de Sitios Web, indicando sus propósitos, competencias generales, así como la estructura que lo compone.
ECBTI-UNAD (2019). Presentación del curso Diseño de Sitios Web. [Archivo de Video]. Recuperado de https://campus01.unad.edu.co/ecbti54/pluginfile.php/2872/mod_folder/intro/PresentacionFinalDSW.mp4.

Propósitos de formación del curso

Se epera que al finalizar el curso, el estudiantes construya sitios web, por medio de un sistema de control de versiones con lenguajes de marca en HTML5 y CSS3, para potenciar las habilidades en el desarrollo de metodologías y herramientas especializadas en línea.

Objetivo del OVI

Ofrecer al estudiante una fundamentación teórica sobre generalidades básicas en el campo del diseño web utilizando sistemas de control de versiones como GIT y la comunidad en línea GITHUB con su cliente para escritorio, potenciando las habilidades en el desarrollo de metodologías y herramientas especializadas en línea, utilizando además de forma correcta los lenguajes de diseño web HTML5 y CSS3, aplicándolos en el diseño de sitios web y Objetos Virtuales de Información (OVI).
