GitHub

Que es:

Es una plataforma de desarrollo colaborativo, donde podemos alojar proyectos utilizando el sistema de control de versiones Git, basicamente es utilizado para la creacion del codigo fuente de alguna aplicacion, pagina, blog etc... Estos proyectos se almacenan en unos repositorios, que es un directorio donde se guardan los archivos del proyecto, este puede estar ubicado en el almacenamiento GitHub o en tu propio ordenador, puedes almacenar archivos de codigo, imagenes, audios y muchas cosas mas.
