HTML

Muchos de nosotros conocemos HTML como un lenguaje de programacion, pero en realidad no lo es sus siglas nos dice que es un Lenguaje de marcas de hipertexto, que hace referencia al lenguaje de marcado, para la elaboracion de paginas web. Es un estandar que conecta la elaboracion de paginas web con sus diferentes versiones, que define una estructura basica y un codigo, que se denomina "codigo HTML" donde se permite alojar en forma de codigo imagenes, videos, juegos, archivos, texto etc... este estandar esta a cargo de World Wide Web (WWW) organizacion dedicada a la estandarizacion de casi todas las tecnologias ligadas a la web.

HTML se escribe en forma de etiquetas rodeadas de corchetes angulares <,> tambien puede escribir hasta un cierto punto, la apariencia de un documento, puede incluir o hacer referencia a un tipo de programa llamado script. El cual puede afectar el comportamiento de navegadores web y otros procesadores de HTML.

Elementos
Los elementos son la estructura basica de HTML estos tienen 2 propiedades basicas, atributos y contenido cada uno de estos dos tiene ciertas restricciones para que sea valido en el codigo HTML, un elemento generalmente tiene una etiqueta de inicio y una etiqueta de cierre.

Un ejemplo de etiqueta de inicio seria: <"etiqueta-de-inicio"> y la de cierre: <"/etiqueta-de-inicio">

https://es.wikipedia.org/wiki/HTML#/media/Archivo:Etiquetas_en_HTML.png

Etiquetas
A continuacion se nombran algunas de las etiquetas utilizadas en codigo HTML, cabe tener en cuenta que un codigo de HTML se debe iniciar con la etiqueta <"html"> y terminar con esta misma <"/html">

<"html"> Define el inicio del documento HTML, lo que le indica al navegador que lo que viene a continuacion debe ser interpretado como codigo HTML. en teoria lo que define el tipo de documento es el DOCTYPE que significa el tag de raiz.

<"script"> incrusta un script en una web o llama a uno mediante un Src="url del script".

<"head"> Define la cabecera del documento, esta cabecera suele contener informacion que no se muestra al usuario, dentro de esta podemos encontrar otras etiquetas como:

<"title"> que define el titulo de la pagina, <"link"> que vincula el sitio para hojas de estilo o iconos, <"style"> que es para colocar estilos internos a la pagina ya sean usando CSS.

<"body"> define el contenido principal o cuerpo del documento.

<"article"> representa una composicion auto-contenida en un documento, pagina, aplicacion o en el sitio que se destina para distribuir de forma independiente.

Otro tipo de etiquetas que nos ayudan mucho en nuestro diseño son: <"h1 a h6"> para encabezados o titulos. <"table"> define una tabla, <"tr"> fila de una tabla. <"a"> define un hipervinculo o enlace. <"img"> agregar imagenes. <"li">,"<"ol">,<"ul"> son etiquetas para listar. Y asi existen aun mas etiquetas para diferentes usos, ya depende de cada programador, darle el uso necesario y correspondiente.
