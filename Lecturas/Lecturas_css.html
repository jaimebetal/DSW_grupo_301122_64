LOS CSS

Los Css (Hoja de estilo en cascada) es un lenguaje de diseño grafico, para definir y crear la presentacion de un documento estructurado escrito en lenguaje de marcado, es muy usado para establecer el diseño visual de los documentos web.

CSS es una tecnologia usada por muchos sitios web para crear paginas visualmente atractivas interfaces de usuario, aplicativos moviles, sirve para darle mas vida a un sitio web. Estos CSS estan diseñados principalmente para marcar la separacion del contenido del documento y la forma de presentacion de este, tanto como las capas, los colores y las fuentes esto busca mejorar la accesibilidad y flexibilidad en la presentacion de un sitio.

En resumen, los CSS son los que mejoran nuestro sitio web, que permite que el usuario se maraville ya sean por sus colores, fuentes o hasta los efectos que estos pueden tener, los CSS son esenciales al momento de programar una pagina web, una aplicacion, entre otros.
